# Student Management System in C

A console-based Student Management System built using C and File Handling.

## Features

- Admin Login System
- Change Password
- Add Student
- View Students
- Search by ID
- Search by Name
- Update Student
- Delete Student
- Statistics Dashboard
- GPA Ranking
- Top 5 Students
- Failed Students List
- Grade System
- Student Report Card
- Export to CSV

## Technologies Used

- C Programming
- Structures
- File Handling
- Sorting Algorithms
- Searching Algorithms

## How to Run

```bash
gcc main.c -o sms.exe
.\sms.exe
```
