# Student Management System

A console-based Student Management System developed in C.

## Features

* Add Student
* View Students
* Search Student
* Update Student
* Delete Student
* GPA Ranking
* Statistics Dashboard
* File Handling

## Technologies Used

* C Programming
* GCC Compiler
* VS Code

## How to Compile

```bash
gcc main.c -o sms.exe
```

## How to Run

```bash
.\sms.exe
```
